# Bullet_Race

#Controls
-A,D turn wheels
-W,S acceleration
-Space break
-X respawn
-1 and 2 change aerodinamic drag

#Mecanics
3 lifes

Win condition: reach the last checkpoint
Lose condition: lose the 3 lifes
